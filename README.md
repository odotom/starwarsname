# odo-starwars-name

## Installation

This package is distributed via npm:

```
npm install odo-starwars-name
```

## Usage

```javascript
const names = require('odo-starwars-name')
const allNames = names.all
const randomName = names.random()
const threeRandomNames = names.random(3)
```
